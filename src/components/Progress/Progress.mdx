# Progress

The Progress component provides visual feedback about the progress of an operation. It can be used to show loading states, upload progress, or any other operation that takes time to complete.

## Basic Usage

```jsx
import { Progress } from './Progress';

// Determinate progress
<Progress value={75} />

// Indeterminate progress
<Progress variant="indeterminate" />

// Circular progress
<Progress variant="circular" value={75} />
```

## Variants

The Progress component comes in four variants:

```jsx
<Progress variant="determinate" value={75} />
<Progress variant="indeterminate" />
<Progress variant="buffer" value={60} bufferValue={80} />
<Progress variant="circular" value={75} />
```

## Colors

The Progress component supports different colors from the theme palette:

```jsx
<Progress color="primary" value={75} />
<Progress color="secondary" value={75} />
<Progress color="error" value={75} />
<Progress color="warning" value={75} />
<Progress color="info" value={75} />
<Progress color="success" value={75} />
```

## Sizes

The Progress component comes in three sizes:

```jsx
<Progress size="small" value={75} />
<Progress size="medium" value={75} />
<Progress size="large" value={75} />
```

## With Value Display

```jsx
<Progress value={75} showValue />
<Progress variant="circular" value={75} showValue />
```

## Custom Thickness

```jsx
<Progress thickness={8} value={75} />
```

## Disabled State

```jsx
<Progress disabled value={75} />
<Progress variant="circular" disabled value={75} />
```

## Buffer Progress

```jsx
<Progress variant="buffer" value={60} bufferValue={80} showValue />
```

## API Reference

### Props

| Name        | Type                                                                    | Default       | Description                             |
| ----------- | ----------------------------------------------------------------------- | ------------- | --------------------------------------- |
| variant     | 'determinate' \| 'indeterminate' \| 'buffer' \| 'circular'              | 'determinate' | The variant to use.                     |
| color       | 'primary' \| 'secondary' \| 'error' \| 'warning' \| 'info' \| 'success' | 'primary'     | The color of the component.             |
| size        | 'small' \| 'medium' \| 'large'                                          | 'medium'      | The size of the component.              |
| disabled    | boolean                                                                 | false         | If true, the progress will be disabled. |
| value       | number                                                                  | 0             | The value of the progress.              |
| bufferValue | number                                                                  | 0             | The buffer value of the progress.       |
| thickness   | number                                                                  | 4             | The thickness of the progress bar.      |
| showValue   | boolean                                                                 | false         | If true, the value will be displayed.   |
