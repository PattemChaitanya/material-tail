# Tooltip

The Tooltip component displays additional information when hovering over an element. It supports different variants, colors, positions, and can include an arrow pointer.

## Basic Usage

```jsx
import { Tooltip } from './Tooltip';

// Basic tooltip
<Tooltip title="This is a tooltip">
  <span>Hover me</span>
</Tooltip>

// Tooltip with arrow
<Tooltip title="With arrow" arrow>
  <span>Hover me</span>
</Tooltip>

// Tooltip with delay
<Tooltip title="Delayed tooltip" enterDelay={200} leaveDelay={200}>
  <span>Hover me</span>
</Tooltip>
```

## Variants

The Tooltip component comes in three variants:

```jsx
<Tooltip title="Standard tooltip" variant="standard">
  <span>Hover me</span>
</Tooltip>
<Tooltip title="Light tooltip" variant="light">
  <span>Hover me</span>
</Tooltip>
<Tooltip title="Dark tooltip" variant="dark">
  <span>Hover me</span>
</Tooltip>
```

## API Reference

### Props

| Name       | Type                                                                                 | Default    | Description                                                    |
| ---------- | ------------------------------------------------------------------------------------ | ---------- | -------------------------------------------------------------- |
| variant    | 'standard' \| 'light' \| 'dark'                                                      | 'standard' | The variant to use.                                            |
| color      | 'primary' \| 'secondary' \| 'error' \| 'warning' \| 'info' \| 'success' \| 'default' | 'default'  | The color of the component.                                    |
| size       | 'small' \| 'medium' \| 'large'                                                       | 'medium'   | The size of the component.                                     |
| position   | 'top' \| 'bottom' \| 'left' \| 'right'                                               | 'top'      | The position of the tooltip.                                   |
| arrow      | boolean                                                                              | false      | If true, adds an arrow to the tooltip.                         |
| enterDelay | number                                                                               | 0          | The number of milliseconds to wait before showing the tooltip. |
| leaveDelay | number                                                                               | 0          | The number of milliseconds to wait before hiding the tooltip.  |
| children   | React.ReactNode                                                                      | -          | The element that triggers the tooltip.                         |
| title      | React.ReactNode                                                                      | -          | The content of the tooltip.                                    |
